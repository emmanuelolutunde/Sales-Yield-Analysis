# Sales-Yield-Analysis

## Tools Used
- Microsoft Excel

## Project Overview
- This project analyzes the sales performance of an outfit who deals in office furniture and appliances.
- The sales were in 2025, between January and May within 4 major cities in Nigeria (lagos, Abuja, Kano and Port harcourt)

## Insights
- Laptop A13 has the highest profit.
- Sales Rep with highest revenue is Peter, hence, highest profit.
- City with the highest sales is Lagos.
- The worst performing month is May.
